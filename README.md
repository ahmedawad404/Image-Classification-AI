# Image Classification System for House Rooms

## Overview
This project is an AI-based image classification system developed to automate the categorization of house room images. It was designed to support a graduation project by processing user-uploaded images efficiently using Deep Learning.

## Tech Stack
* **Framework:** PyTorch
* **Backbone:** ResNet-50 / ResNet-101 (Transfer Learning)
* **Language:** Python
* **Tools:** Google Colab, GitHub

## Key Features
* **Data Augmentation:** Implemented advanced transformations to improve model generalization.
* **Evaluation:** Comprehensive testing using Confusion Matrix and Classification Reports.
* **Optimization:** Fine-tuned pretrained models to achieve high accuracy in specialized datasets.
